# Human-Detection-Real-Time

It can detect successfully humans in real time 

Run the main.py file only.

Detect humans from a video - Python main.py -v ‘Path_to_video’
Detect humans from a photo - Python main.py -i ‘Path_to-image’
Detect humans from webcam - Python main.py -c True
Save the output file - Python main.py -c True -o ‘file_name’

python version - 3.6
opencv - cv2 version - 4.4

rest libraries are custom made python libraries.
